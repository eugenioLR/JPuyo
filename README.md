# JPuyo
puyo-puyo made in Java  
  
Controls:  
Player 1:  
* d: move right  
* a: move left  
* q: rotate left  
* e: rotate right  
* s: fall once  
* w: drop to the bottom  
  
Player 2:  
* →: move right  
* ←: move left  
* n: rotate left  
* m: rotate right  
* ↓: fall once  
* ↑: drop to the bottom
